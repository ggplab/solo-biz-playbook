BRAND NAMING PROCESS

Brand Naming Process

후보 단어를 만들기 전에 헌법과 톤을 먼저 닫는 5단계 절차입니다. 발산은 3단계, 결정은 5단계 끝에서 일어납니다.

Criteria before candidates

5축 헌법과 톤 4룰을 먼저 닫지 않으면, 어떤 좋은 단어가 나와도 결정권자가 매번 다른 잣대로 깎아내고 결정이 미뤄집니다.

Diverge late, compress hard

후보 발산은 3단계까지 미루고, 그다음 SMILE / SCRATCH 두 체크리스트로 5개 이하까지 단번에 줄입니다.

Test before commit

등기·상표·도메인·다국어 4중 검수에 잠재 고객 5명 받아 적기 테스트까지 통과해야 도메인 결제·로고 외주를 시작합니다.
