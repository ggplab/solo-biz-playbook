# 원칙 03 — 부의 사다리 4단 매핑

> "시간을 팔 것인가, 결과물을 팔 것인가, 시스템을 팔 것인가. **같은 1시간에 무엇이 남느냐**가 사업 모델의 본질이다."

Based on: Nathan Barry, "Ladders of Wealth Creation"

---

## 한 줄 정의

모든 상품·수익원을 **4단 사다리** 중 하나에 매핑하고, **시간 흐름에 따라 상위 단계로 비중을 옮긴다.**

| 단계 | 이름 | 특징 | 시간↔돈 구조 |
|---|---|---|---|
| 🪜 1단 | **시간교환** (Trading Time) | 멘토링·컨설팅·자문 | 1시간 일하면 1시간치 돈 |
| 🪜 2단 | **서비스** (Providing a Service) | 기업 강의·VOD 제작·프로젝트 용역 | 1건 납품하면 패키지 단가 |
| 🏗️ 3단 | **자산** (Building Assets) | 온라인 강의·저서·템플릿·앱 | 한번 만들면 반복 수입 |
| ⚙️ 4단 | **시스템** (Owning Systems) | 유료 커뮤니티·구독·플랫폼 | 운영되는 동안 복리 수입 |

---

## 왜 중요한가

**1인 사업자의 구조적 함정**

대부분의 1인 사업자는 1단(시간교환)에서 시작하고, **평생 1단에 머문다.** 이유:
- 1단이 당장 제일 현금이 잘 돈다
- 상위 단계는 **선행 투자** (제작 기간·무수익 기간)가 필요하다
- "자산을 만들기에는 바빠서 시간이 없다"는 악순환

**사다리 프레임의 의도**

- 지금 어느 단계에 있는지 **명시적으로 보게** 한다.
- 상위 단계로의 이동을 **숫자 목표** 로 박는다. (자산 비중 %)
- 1단만으로는 **번아웃 전에 사업이 끝난다** 는 걸 수치로 이해하게 한다.

---

## 각 단계의 특성

### 🪜 1단 — 시간교환 (Trading Time)

- **예시**: 1:1 멘토링, 컨설팅, 자문, 개인 과외
- **장점**: 즉각 현금, 고객 신호 직접 수집, 복잡한 제작 불필요
- **단점**: 확장 불가 (시간은 한정), 병가·휴가 = 매출 0
- **적정 비중**: 초기 40~60%, 3년차 5~15%로 감축 목표
- **전략 포인트**: **소재 발굴 채널**로 활용. 반복 질문 = 상위 단계 상품 아이디어.

### 🪜 2단 — 서비스 (Providing a Service)

- **예시**: 기업 강의, VOD 제작 수주, 프로젝트 용역
- **장점**: 1단보다 단가 높음, 레퍼런스 축적, 전문성 증명
- **단점**: 여전히 시간 비례, 클라이언트 커스텀 공수, 한 번 납품하면 끝
- **적정 비중**: 현재 주력(~90%) → 3년차 35~45%로 조정
- **전략 포인트**: **자산(3단)의 원재료 수집소**. 계약서에 IP 이중활용 조항 필수.

### 🏗️ 3단 — 자산 (Building Assets)

- **예시**: 온라인 강의, 저서, 유료 템플릿·스타터킷, 유료 전자책, 앱
- **장점**: 한 번 만들면 반복 수입, 시간 독립성, 브랜드 자산화
- **단점**: 제작 기간 동안 무수익, 성공 불확실, 초기 마케팅 공수
- **적정 비중**: 1년 25% → 2년 40% → 3년 55% 로드맵
- **전략 포인트**: **2단에서 만든 교안·예제·사례**를 자산화. 백지에서 시작 금지.

### ⚙️ 4단 — 시스템 (Owning Systems)

- **예시**: 유료 커뮤니티, 구독 멤버십, SaaS, 유료 플랫폼
- **장점**: 복리 수익, 운영 레버리지, 네트워크 효과
- **단점**: 운영 부담 지속, 기술·커뮤니티 인프라 필요
- **적정 비중**: 초기 0% → 3년차 5~10%
- **전략 포인트**: 너무 일찍 진입하면 **1인 운영 한계**에 부딪혀 모든 단계가 망가진다. 자산 축이 자리잡은 뒤.

### 설계 체크리스트
- [ ] 1단 비중이 3년차에 **15% 이하**로 내려가 있는가?
- [ ] 3단 비중이 3년차에 **50% 이상**이 되는가?
- [ ] 2단은 버리는 게 아니라 **자산의 원재료 공급원**으로 유지되는가?
- [ ] 4단은 **3단이 자리잡은 이후** 에 진입하는가?

## 운영 규칙

1. **매월 매출을 4단으로 분류** 해서 비중을 기록한다.
2. **IP 시간(자산 제작용 비수익 시간) 확보율** 을 선행지표로 추적한다. (목표: 월 120h)
3. 수락/거절 의사결정 시 **"이것이 어느 단계로 가는가?"** 를 명시적으로 묻는다.
4. **2단 수주에 포함된 IP 권리** 는 계약 단계에서 반드시 협상 (이중활용 가능성).
5. **분기 회고** 에서 비중이 로드맵 대비 2%p 이상 지연되면 원인 분석 + 다음 분기 조정.

## 흔한 반론과 응답

**"자산 만들 시간이 안 나는데."**
→ 시간이 안 나는 게 아니라 **IP 시간을 먼저 블록킹하지 않아서** 다. 투자 모드(수입활동 40h / IP 120h) 기본 배분으로 스케줄부터 뒤집는다.

**"만든 자산이 안 팔리면?"**
→ 2단에서 **검증된 소재** 로 만들면 실패율이 낮다. 1단(멘토링 반복 질문) → 2단(강의 만들고 반응 확인) → 3단(VOD화)의 검증 동선을 타라.

**"시간교환(1단)을 아예 버려야 하나?"**
→ 버리지 않는다. **비중을 낮추되 유지**. 1단이 **시장 신호 수집 채널** 이자 **대체 불가 관계망**이다. 단, 시급 방어선 아래 1단 수주는 거절.

**"나는 4단(커뮤니티·구독)부터 바로 시작하고 싶은데."**
→ 1~2단을 통해 **brand·audience·운영 근육**이 없는 상태의 4단은 거의 실패한다. 사다리는 순서가 있다.

---

## 다른 맥락에서의 적용

- **크리에이터/유튜버**: 1단(코칭) → 2단(브랜드 스폰서십 프로젝트) → 3단(유료 강의·굿즈) → 4단(멤버십)
- **디자이너**: 1단(1:1 컨설팅) → 2단(프로젝트 수주) → 3단(유료 템플릿·프리셋) → 4단(Figma 플러그인 구독)
- **작가**: 1단(첨삭·강연료) → 2단(원고 용역) → 3단(저서·강의) → 4단(유료 뉴스레터)
