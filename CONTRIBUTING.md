# Contributing

한국 1인 사업자 린캔버스 **컬렉션**을 만드는 게 이 레포의 장기 목표입니다. 당신의 캔버스를 익명화해서 PR 보내주시면 환영합니다.

## 사례 기여 방법

1. [`template/lean-canvas.md`](template/lean-canvas.md)를 복사해 자기 버전을 채운다.
2. **민감정보를 마스킹**한다. 공개 가이드:
   - 절대 금액 → 범위 또는 배수 ("월 500~800만대", "매출 기본 베이스의 3배")
   - 고객사 실명 → 유형으로 ("대기업 3곳·커머스 플랫폼 등")
   - 협상 중인 계약·비공개 수치는 빼기
   - **유지해도 되는 것**: 비중(%), 전략, 원칙, 이론 레퍼런스
3. `examples/` 에 `<본인식별자>.md` 파일로 추가한다. (익명 닉네임도 OK)
4. PR 제목: `Add example: <식별자> (<업종 한 줄>)`
5. PR 본문에 다음을 포함: 업종, 사업 연차, 본인이 강조하고 싶은 "배운 점" 1~3개

## 원칙 문서 기여

`docs/principles/` 아래 원칙은 본인 사례 기반으로 쓰여 있습니다. 반례나 다른 업종 맥락의 조정안을 PR로 보내주시면:
- 원문은 유지하고, 섹션 끝에 `### 다른 맥락: <업종/상황>` 블록으로 추가합니다.

## 번역

v2에서 영어 번역본을 계획 중입니다. 기여하고 싶으시면 먼저 Discussions에 의사를 남겨주세요.

## 코드·자동화는 없습니다

이 레포는 문서 전용입니다. 스크립트·빌드·CI 없음. 오타·링크 깨짐 수정 PR 환영.

## 질문·토론

구체 PR이 아니라 질문이라면 [Discussions](../../discussions) 를 사용해주세요.
